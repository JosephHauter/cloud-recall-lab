# Security Policy

Cloud Practitioner Practice Hub is a static app with no backend, accounts, or server-side database. Progress is stored locally in the learner's browser.

## Reporting

Please open a GitHub issue for security or privacy concerns after the repository is public. If the issue involves a sensitive exploit, contact Joseph Hauter through LinkedIn instead of posting public reproduction details.

## Scope

- Client-side JavaScript issues
- Content Security Policy regressions
- Unexpected external scripts, trackers, or network calls
- Local progress backup/import problems

## Out of Scope

- AWS certification exam disputes
- Requests for actual certification exam questions, answer dumps, or confidential assessment material
- Issues caused by browser extensions or modified local files

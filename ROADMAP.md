# Roadmap

This roadmap favors launch-ready simplicity over adding features that create privacy, maintenance, or trust problems.

## Before Publishing

- Create the GitHub repo named `cloud-practitioner-practice-hub`.
- Push the current `main` branch and enable GitHub Pages.
- Test the live HTTPS URL on desktop and mobile.
- Verify the PWA install prompt and offline reload from the live URL.
- Share the LinkedIn launch post with a screenshot or short screen recording.

## Newly Added Launch Polish

- Cloud Recall Lab public name to avoid colliding with existing practice-hub projects.
- Serverless service map and service-index badges.
- Local score-card image download for sharing mock results.
- Compatibility copy for modern browsers, mobile use, and local progress.

## High-Value Next Features

- Question report button that opens a prefilled GitHub issue.
- Keyboard shortcuts for next, previous, flag, and answer choices.
- Print-friendly results page for people who like offline review.
- Bookmarked questions for manual review.
- Study streak view based only on local browser history.
- Optional privacy-friendly analytics after the live privacy copy is updated.

## Content Quality

- Re-audit every mock exam against the current AWS CLF-C02 exam guide before large promotions.
- Add more per-option rationales where explanations are too short.
- Keep answer distribution balanced across A, B, C, and D.
- Track dated AWS wording changes in GitHub issues.

## Later

- Additional AWS certifications after the CLF-C02 app is stable.
- Additional interface languages after the current beta copy is reviewed.
- GitHub Discussions and a small maintainer guide if contributors show up.
